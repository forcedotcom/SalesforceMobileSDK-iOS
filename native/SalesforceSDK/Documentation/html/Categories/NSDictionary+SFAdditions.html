objectAtPath:

Get object from NSDictionary with “/” separated path.

- (id)objectAtPath:(NSString *)path

Parameters

path

Path for the object to retrieve, use “/” to separate between levels. For example, root/child/valueKey will retrieve value from the root NSDictionary object to it’s child dictionary’s value with key “valueKey”

Discussion

Get object from NSDictionary with “/” separated path.

This function is similar to the built-in valueForKeyPath function except it handles special value like NSNULL and in the NSDictonary element value*
