parseResponse

@property (nonatomic,
    assign) BOOL parseResponse;
Discussion

Whether or not to parse the response data as structured JSON data. Defaults to `YES`. If set to `NO`, response data will be returned as binary data in an `NSData` object.

path

@property (nonatomic,
    strong) NSString *path;
Discussion

The path of the request. For instance, "" (empty string), "v22.0/recent", "v22.0/query". Note that the path doesn't have to start with a '/'. For instance, passing "v22.0/recent" is the same as passing "/v22.0/recent".

WARNING:

Do not pass URL encoded query parameters in the path. Use the `queryParams` property instead.

queryParams

@property (nonatomic,
    strong) NSDictionary *queryParams;
Discussion

The query parameters of the request (could be nil). Note that URL encoding of the parameters will automatically happen when the request is sent.

requiresAuthentication

@property (nonatomic,
    assign) BOOL requiresAuthentication;
Discussion

Whether or not this request requires authentication. If YES, the credentials will be added to the request headers before sending the request. If NO, they will not.
