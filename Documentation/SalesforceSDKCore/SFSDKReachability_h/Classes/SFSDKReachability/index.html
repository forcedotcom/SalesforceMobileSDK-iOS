connectionRequired

Determines whether a connection is required for network access. 

- (BOOL)connectionRequired;
Return Value

YES if further network connections are required, NO otherwise.

Discussion

WWAN may be available, but not active until a connection has been established. WiFi may require a connection for VPN on Demand.

currentReachabilityStatus

The current reachability status. 

- (SFSDKReachabilityNetworkStatus)currentReachabilityStatus;
Return Value

SFSDKReachabilityNetworkStatus representing the current reachability status.

reachabilityForInternetConnection

Checks whether the default route is available. 

+ (instancetype)reachabilityForInternetConnection;
Return Value

An instance of SFSDKReachability configured to reach the network via the default route.

Discussion

Should be used by applications that do not connect to a particular host.
