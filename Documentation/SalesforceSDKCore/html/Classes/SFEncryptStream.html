SFEncryptStream Class Reference

Inherits from NSOutputStream
Conforms to SFCryptChunksDelegate

Overview

SFEncryptStream is an output stream that encrypts data before it is written out. SFCryptChunks is used to perform the encryption.

– setupWithKey:andInitializationVector:

Setup for encryption. You must call this method before using the stream.

- (void)setupWithKey:(NSData *)key andInitializationVector:(nullable NSData *)iv

Parameters

key

the cipher key.
iv

the initialization vector, must be size of SFCryptChunksCipherBlockSize.
