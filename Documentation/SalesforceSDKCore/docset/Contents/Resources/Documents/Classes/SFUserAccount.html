Overview

– transitionToLoginState:

Attempts to transition from the current login state to the new login state.

- (BOOL)transitionToLoginState:(SFUserAccountLoginState)newLoginState

Parameters

newLoginState

The new login state to set.

Return Value

YES if the login state change was successful, NO otherwise.

Discussion

Only certain state transitions are allowed. For example, you cannot transition from SFUserAccountLoginStateNotLoggedIn to SFUserAccountLoginStateNotLoggingOut, but you can transition from SFUserAccountLoginStateNotLoggedIn to SFUserAccountLoginStateLoggedIn.
