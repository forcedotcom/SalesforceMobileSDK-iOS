changePasscode:

- (void)changePasscode:(NSString *)newPasscode;
Parameters
newPasscode

The new passcode to change to. If nil or empty, this method will unset the existing passcode.

Discussion

Change the current passcode. This method serves as an entry point for managing the change or removal of a passcode, notifications of the change, etc. The setPasscode method, by comparison, handles the internals of actually setting a new passcode value.
