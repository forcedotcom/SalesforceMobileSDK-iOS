isUserAnonymous:

+ (BOOL)isUserAnonymous:(SFUserAccount*)user;
Discussion

Returns YES if the specified user is anonymous. Note: an anonymous user is a user that doesn't require credentials towards a server.

isUserTemporary:

+ (BOOL)isUserTemporary:(SFUserAccount*)user;
Discussion

Returns YES if the specified user is temporary. Note: a temporary user is created when a new user is requested, for example during the login into a new org, and is replaced by the real user once the login is finished.
